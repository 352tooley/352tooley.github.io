# Payloads

Place GoldHEN BinLoader-compatible payload binaries here.

## Expected files

| Filename            | Firmware |
|---------------------|----------|
| `payload-12.50.bin` | FW 12.50 |
| `payload-12.52.bin` | FW 12.52 |

## Sources

Payloads are firmware-specific and must be obtained from the PS4 homebrew community:

- GoldHEN: https://github.com/GoldHEN/GoldHEN
- BinLoader: https://github.com/Al-Azif/ps4-payload-sdk

**Do not commit payload binaries to this repository.**
Add `*.bin` to `.gitignore` if needed.
