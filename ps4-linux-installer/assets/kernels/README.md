# Kernels

Place precompiled Linux kernel images here.

## Expected files

| Filename         | Distro      |
|------------------|-------------|
| `bzImage-arch`   | Arch Linux  |
| `bzImage-xubuntu`| Xubuntu     |

## Sources

PS4-compatible Linux kernels (patched for PS4 hardware):

- https://github.com/fail0verflow/ps4-linux
- https://ps4linux.com/kernels/

Build from source or download precompiled binaries from the PS4 Linux community.

**Do not commit large kernel binaries to this repository.**
Use Git LFS or provide a download script if needed.
